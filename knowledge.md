# 🌱 AgroNexus Knowledge Base

## [KB-001] Temperatura óptima

La temperatura ideal para la mayoría de cultivos se encuentra entre 20°C y 30°C.

- 20°C – 30°C → crecimiento óptimo
- 30°C – 35°C → crecimiento aceptable con monitoreo
- > 35°C → estrés térmico
- < 15°C → crecimiento lento

Impacto:
- Altas temperaturas aumentan evaporación
- Puede requerirse riego adicional

---

## [KB-002] Humedad del ambiente

La humedad relativa ideal está entre 40% y 70%.

- 40% – 70% → rango óptimo
- 30% – 40% → humedad baja
- < 30% → sequedad crítica
- > 80% → riesgo de hongos

Impacto:
- Humedad baja → deshidratación
- Humedad alta → proliferación de enfermedades

---

## [KB-003] Nivel de luz

El nivel de luz adecuado está entre 60 y 80.

- 60 – 80 → óptimo
- 40 – 60 → aceptable
- < 40 → deficiencia lumínica
- > 85 → exceso de radiación

Impacto:
- Luz alta + temperatura alta → estrés hídrico
- Luz baja → crecimiento lento

---

## [KB-004] Diagnóstico combinado

Reglas clave:

- Temperatura alta + humedad baja → sequedad acelerada
- Humedad alta + baja luz → riesgo de hongos
- Temperatura alta + luz alta → estrés hídrico
- Humedad baja constante → necesidad de riego

---

## [KB-005] Recomendaciones prácticas

- Regar cuando humedad < 40%
- Reducir exposición solar si luz > 85
- Aumentar ventilación si humedad > 80%
- Monitorear cambios bruscos

---

## [KB-006] Análisis de tendencia

No solo se deben analizar valores actuales.

Se debe considerar:

- Si la humedad está bajando progresivamente
- Si la temperatura está subiendo rápidamente
- Cambios bruscos en luz

Interpretación:
- Tendencias son más importantes que valores aislados
